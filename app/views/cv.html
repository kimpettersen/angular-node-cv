This web application is written in Angular JS which is communicating with an API written in Node JS using MongoDB as a database. The application has both unit and end-to-end tests. You are encouraged to clone the project from Github and play with it.

This application lets you perform CRUD operations from the browser. The access is normally restricted, but you can perform read operations without being logged in.
These are the available resources:
  • /api/education
  • /api/experience
  • /api/me
  • /api/bucketlist
For documentation and a post-mortem discussion, visit the documentation on Github
